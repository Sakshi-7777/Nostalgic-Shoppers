# Nostalgic Shoppers

Nostalgic Shoppers is a responsive, multi-page e-commerce storefront designed for a personalized shopping experience. Built using a clean, standard front-end stack of HTML5, CSS3, and Vanilla JavaScript, the project focuses on delivering a smooth user interface for browsing and purchasing customized apparel, home décor, and accessories.

The application is currently deployed and accessible via Vercel at: [https://nostalgic-shoppers.vercel.app](https://nostalgic-shoppers.vercel.app)

---

## Core Features

### Dynamic Storefront and Product Management
The project implements a full catalog system where users can browse through various categories:
*   **Curated Collections:** Specialized categories including Summer Apparel, Anime-themed Hoodies, and Handcrafted Jewellery.
*   **Product Interaction:** Individual product pages feature a custom-built image gallery switcher. This allows users to inspect products from multiple angles before adding them to their cart.
*   **Cart System:** A dedicated cart interface enables users to review their selected items and manage quantities effectively.

### Modern User Interface
*   **Fully Responsive Design:** The layout adapts seamlessly to desktops, tablets, and mobile devices. A custom-built hamburger menu handles navigation on smaller screens.
*   **Promotional Systems:** Strategic placement of banners for limited-time offers, seasonal sales (up to 70% off), and "Buy 1 Get 1" deals to drive engagement.
*   **Informational Pages:** Comprehensive "About" and "Blog" sections to communicate brand values and the latest store updates.

### Communication and Support
*   **Contact and Feedback:** A structured contact page with office locations (based in Bhubaneswar, Odisha), direct contact info, and integrated Google Maps support.
*   **Newsletter Integration:** A footer-based subscription system for capturing leads and keeping customers updated on New Arrivals.

---

## Technical Architecture

The project is built with a focus on web standards and performance without the overhead of heavy frameworks:

*   **Structure (HTML5):** Semantic markup ensures better SEO and accessibility across the multi-page architecture.
*   **Styling (CSS3):** Custom layouts using Flexbox and Grid for responsive positioning, coupled with smooth transitions for hover effects and modal interactions.
*   **Logic (Vanilla JavaScript):** Handles DOM manipulation for the navigation drawer, product image switching, and basic UI state management.
*   **Deployment:** Configured for Vercel with a `vercel.json` file for optimized hosting.

---

## Project Structure

The repository is organized logically for easy maintenance:

*   **index.html:** High-traffic landing page with featured products and hero sections.
*   **shop.html:** The primary product listing page with pagination placeholders.
*   **shop_product.html:** Template for detailed product views and image galleries.
*   **cart.html:** Shopping cart management interface.
*   **blog.html / about.html / contact.html:** Supporting pages for brand content and support.
*   **style.css:** Central stylesheet for global design tokens and layout rules.
*   **script.js:** Core interaction logic.
*   **imgs/:** Assets directory containing product imagery and UI icons.

## Getting Started

### Local Development
1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Sakshi-7777/Nostalgic-Shoppers.git
    ```
2.  **Open the project:**
    Navigate to the root directory and open `index.html` in any modern web browser. Using a local development server (like VS Code's Live Server) is recommended for the best experience.

### Deployment
The project is set up for automatic deployment via Vercel. Simply connect your GitHub repository to a Vercel project to enable continuous integration.
